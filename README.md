# Aaron Strasler — MRI Technologist

Personal portfolio and resume site for Aaron Strasler, an MRI Technologist with clinical experience in diagnostic imaging.

## Live Site

Visit the live site at: [https://astrasler.github.io/AaronStrasler/](https://astrasler.github.io/AaronStrasler/)

## About

This repository hosts Aaron's professional online resume, built with HTML and CSS and deployed via GitHub Pages. The site includes:

- Professional summary and work experience
- Education and clinical training history
- Technical skills and certifications
- Contact information

## Technologies

- HTML5 / CSS3
- GitHub Pages (deployment)
- GitHub Actions (CI/CD with CodeQL security scanning and stale issue management)
